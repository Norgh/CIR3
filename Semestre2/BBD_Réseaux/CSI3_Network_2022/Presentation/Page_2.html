Compréhension - QCM - Présentation

Présentation  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           
Cette fonctionnalité vous permet de suivre chaque activité.

Gestion
Dans le bloc "Compréhension - QCM" situé en bas de la première page de chaque chapitre de cours, des icônes , et vous permettent de gérer ce suivi.


1 - Activer
Vous pouvez activer cette option en cliquant sur l'icône . Un tableau de bord s'ajoute alors en bas de cette page.

2 - Désactiver
Vous pouvez désactiver cette option sans perdre pour autant les informations collectées en cliquant sur l'icône . Vous pourrez la réactiver ultérieurement. Le tableau de bord est masqué.

3 - Reset
Vous pouvez supprimer les informations collectées en lien avec le chapitre du cours en cliquant sur l'icône .


Reset ALL database !!!
Vous pouvez supprimer TOUTES les INFORMATIONS collectées de l'ensemble du cours, y compris TOUTES les NOTES en cliquant sur l'icône .

Cette option n'est accessible que depuis la première page de cette présentation. ATTENTION : la base de données locale à votre navigateur web sera complètement vidée.

Ces informations seront stockées dans la base de données locale à votre navigateur web, il faudra donc éviter d'en changer.
Utilisation  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           
Activité
Vous pouvez indiquer qu'une activité nécessite l'aide d'un enseignant ou d'y revenir ultérieurement en cliquant sur l'icône :
- Le compteur associé est surligné et s'incrémente.
- Un encart "bleu" lié à l'activité s'ajoute dans le tableau de bord de la première page.

La validation de cette option se fait en cliquant sur l'icône , l'encart associé disparait du tableau de bord.


QCM
De même pour un QCM :
- Deux compteurs et totalisent en plus les résultats.
- Si l'une des questions n'est pas validée, un encart "rouge" lié à l'activité s'ajoute dans le tableau de bord de la première page.

La validation de cette option se fait aussi en cliquant sur l'icône , l'encart associé disparait du tableau de bord.


Tableau de bord
Le tableau de bord qui regroupe tous les encarts, vous permet de revenir sur chaque point en attente.

Au niveau de chaque activité, un lien sur "Compréhension" et "QCM" vous permet de revenir sur le tableau de bord et de sélectionner les points un par un.
