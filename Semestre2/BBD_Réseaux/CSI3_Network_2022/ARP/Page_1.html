Clarification - Protocole ARP

Quelle adresse pour quel usage ?  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           
L'adresse IP permet d'identifier les postes au niveau international, alors que l'adresse Ethernet permet d'acheminer les messages au niveau local.

Ces adressages pourraient jouer un rôle concurrentiel, mais leur fonction est bien distincte et complémentaire.

Le rôle du protocole ARP sera de maintenir la correspondance entre ces deux référentiels.
