Définitions - Protocole ARP

Définitions  
ARP

Address Resolution Protocol

Poste local

Ses propres paramètres sont utilisés pour identifier l’expéditeur.

Remettre un message

Le message sera encapsulé dans un datagramme IP, lui-même encapsulé dans une trame Ethernet.

Tous les postes

L’utilisation d’un broadcast permet de questionner l’ensemble des postes en n’envoyant qu’une seule trame Ethernet.

Requêtes

Le protocole ARP contribue à renseigner les champs relatifs au destinataire pour lesquels les valeurs ne sont pas connues localement.

Référentiels

Chaque protocole assure des fonctions spécifiques, certaines liées au niveau local (Ethernet) et d’autres liées au niveau international (IP).

Résolution

Un mécanisme d’interrogation permet d’obtenir l’adresse Ethernet du propriétaire d’une adresse IP.

Cette opération se déroule sur un réseau local et n’est valide que dans ce périmètre.

Cache

Un cache est une table de correspondance dont les informations ont une durée de vie limitée.

Dans le cas de l’ARP, la correspondance est faite entre des adresses IP et des adresses Ethernet.

Niveau international

Ce niveau constitue un unique référentiel mondial.

Il s’agit d’établir un itinéraire entre les points de communication ainsi identifiés (@IP).

Les technologies utilisées pour acheminer les informations n’entrent pas dans cette préoccupation.

Niveau local

Ce niveau regroupe des équipements locaux dont le nombre ne dépasse généralement pas quelques centaines.

Ils peuvent communiquer grâce à un même protocole sur des liens filaires ou sans fil.

Aucune politique de sécurité ne n’applique entre eux.

Réponse

Pour mettre à jour le cache ARP du correspondant, une réponse ARP est renvoyée suite à la réception d’une requête ARP.

Régulière

Un compromis est défini pour maintenir une information à jour sans pour autant surcharger le réseau local.

Dynamiquement

Le système maintient automatiquement la correspondance pour prendre en compte tout changement.

Requête

Pour obtenir l’adresse Ethernet, une requête ARP est générée avec l’adresse IP correspondante.

Temps limité

Il s’agit d’un délai de l’ordre de 10 minutes.
