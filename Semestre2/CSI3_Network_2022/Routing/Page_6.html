Définitions - Routage entre réseaux

Définitions  
Passerelle

Une adresse IP est désignée pour être le prochain point de passage intermédiaire pour acheminer le datagramme.

Inefficace

La consultation d’une table de routage s’effectue à chaque envoi de datagramme. Plus le nombre de destinations augmente, plus le temps de consultation sera long.

D'autres classes IP

Un réseau local associé à une classe IP ne peut être accédé s’il n’est pas déclaré dans les tables de routage des équipements qui le séparent des autres réseaux locaux.

Intersection

Le poste source et les routeurs constituent des intersections sur le trajet vers le destinataire.

A chaque intersection, une table de routage indique la direction à prendre.

Table de routage

Les indications de direction constitueront une liste d’entrées dans la table. Chaque message déclenchera le parcours de cette table afin de déterminer l’itinéraire qui lui conviendra d’emprunter.

Regroupement

La modification de la valeur d’un masque dans une table de routage permet d’ajuster la couverture d’une déclaration, ce qui conduit à une réduction du nombre de lignes nécessaires.

Statique

La table de routage a été initialisée manuellement et son contenu ne peut être modifié que par une intervention humaine.

Dynamique

Un protocole de routage permet de calculer de façon optimale les itinéraires et de renseigner automatiquement le contenu des tables de routage.
