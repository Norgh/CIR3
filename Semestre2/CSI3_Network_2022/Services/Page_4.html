DNS - Services

Cours  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           


L'adressage d'un service sur un poste nécessite de connaitre son adresse IP.

Afin de simplifier l'identification d'un service et de masquer une éventuelle reconfiguration de l'adressage IP d'un site, un mécanisme d'annuaire a été mis en oeuvre.

Le service DNS permet d'obtenir automatiquement l'adresse IP à partir d'une URL fournie sous la forme d'un FQDN.
